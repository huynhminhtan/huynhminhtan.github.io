## Hướng dẫn

- Cài đặt NVM: `curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash`
- Cài Nodejs theo version mong muốn: `nvm install 8.9.0`
- Kiểm tra: `node -v`
- Trang xem version Nodejs: https://nodejs.org/dist/
